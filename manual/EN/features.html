VenvHub Pro

The ultimate developer tool for managing, orchestrating, and distributing Python applications.

Version v2.5.14 | Created by: Seal and Lube, spol. s r.o.

Why VenvHub Pro?

VenvHub Pro is not just another GUI wrapper for the command line. It is a comprehensive IDE-agnostic environment that solves real-world problems for Python developers – from "Dependency Hell" and carrying code on USB sticks, to microservices orchestration and packaging into .exe files. It replaces 5 different tools with one smooth, asynchronous application with instant response (in-memory cache).

📁 Separated Architecture

No more code clutter! VenvHub Pro physically separates your source code from virtual environments. Your projects stay clean, while all Venvs are centrally managed in a dedicated Venv Hub folder. If you delete a Venv, your code is safe.

💻 Isolated VS Code Profiles

Open VS Code and IntelliSense works immediately. VenvHub silently synchronizes settings.json, assigns Venvs, Tasks, and Keybindings. Additionally, it offers fully isolated user profiles – each developer or project can have separate extensions and editor settings.

📦 Dual-Scan Pip & UV Autopilot

Extremely fast package manager. It uses parallel scanning to detect outdated libraries. Revolutionary feature: When using the uv installer, VenvHub can perform a deep check after an update and automatically fix dependency conflicts (Auto-Fix) by precisely downgrading broken packages.

🔄 Venv Orchestrator & Watchdog

Group Venvs (Frontend, Backend, DB) into Containers. VenvHub launches them in precise order using timers and a signaling "Hook" to guarantee startup. The Watchdog monitors process PIDs and intelligently restarts (Respawn) them up to 3 times in case of failure.

🛠 PyInstaller & Auto-Injector

Visual creation of .exe files. Not only does it bundle the uv.exe installer and generate a "Birth Certificate" (JSON metadata) with the Python version, but thanks to the Auto-Injector, it automatically reads linked local packages and dynamically inserts their source code and assets into the build.

🐍 Local Embeddable Runtimes

Download "Local" Python versions from ZIP archives without needing to install them into Windows (no registry or PATH modifications). VenvHub automatically "hacks" and repairs them (installs Pip, unlocks site-packages in the ._pth file) so they behave like full installations.

🧳 Portability Protection (Portable)

Coding on a USB drive? If the drive letter changes on another PC, the Venv Validator automatically repairs all paths. Simultaneously protects your code: The System Listener monitors the computer's Hardware ID and prevents incompatible system Venvs from running on unauthorized machines.

🎨 Dual-Theme & AI Modding

Modern UI built on a robust bridge for PyQt6/PySide6, running completely asynchronously (doesn't block during installations). It includes a system of built-in and user-defined QSS skins. Have an AI recolor your skin and import it back into the application with one click!
